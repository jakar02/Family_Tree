Projekt ma na celu wykorzystanie grafowej bazy danyc Neo4j. W naszym projekcie baza zawiera relacje typu: PARTNER i HAS_CHILD.
Aplikacja umożliwa tworzenie nowych drzew genealogicznych oraz ich edycję bez potrzeby logowania.
